# 😂 Joke Generator

A modern Joke Generator web application built using **HTML, CSS, JavaScript**, and the **API Ninjas Joke API**.

## 🚀 Features

- 🎲 Get Random Jokes
- 😂 Get Joke of the Day
- Beautiful Responsive UI
- Glassmorphism Design
- Fetch API with Async/Await
- Error Handling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- API Ninjas Joke API
